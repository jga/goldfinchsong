===============================
**goldfinchsong** Documentation
===============================

The **goldfinchsong** package helps users easily post tweets from an image library.

Why?
----

If you're a twitter user that shares image content such as
landscape pictures or data charts, you may desire to automate
the posting of content from your image library.
